## Deployment

This site is a single self-contained `index.html` file with all images embedded as base64. No build step or dependencies required.

To deploy on GitHub Pages:
1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch**
4. Select **main** branch, **/ (root)** folder
5. Save — the site will be live within a minute
